# 11_Deep_Learning

一个基于 `PyTorch` 的深度学习基础练习仓库，内容从张量操作、自动微分、神经网络基础，一直到 `ANN` 和 `CNN` 的入门案例。

## 仓库内容

- 张量创建、索引、形状变换、拼接
- `numpy` 与 `torch` 相互转换
- 自动微分、反向传播、梯度更新
- 激活函数、损失函数、参数初始化
- 优化器、学习率衰减、`Dropout`、`BatchNorm`
- 全连接神经网络分类案例
- 卷积神经网络图像分类案例

## 目录说明

### `Day01`

张量基础：

- 张量的基本创建方式
- 线性张量和随机张量
- 全 0、全 1、指定值张量
- 指定类型张量

### `Day02`

张量常用操作：

- `numpy` 和 `torch` 之间转换
- 基本运算、点乘、矩阵乘法
- 常用运算函数
- 索引操作、形状操作、张量拼接
- 自动微分入门

### `Day03`

训练基础与激活函数：

- `detach()` 用法
- 自动微分的实际使用
- `PyTorch` 线性回归案例
- `Sigmoid`、`Tanh`、`ReLU`、`Softmax`

### `Day04`

神经网络搭建基础：

- 参数初始化
- 自定义 `nn.Module`
- 前向传播流程
- 多分类、二分类、回归任务常见损失函数

### `Day05`

训练技巧与 `ANN` 入门：

- 梯度下降与常见优化器
- 学习率衰减策略
- `Dropout`
- `BatchNorm`
- 手机价格分类案例

### `day06`

`ANN` 完整训练流程和图像基础：

- 手机价格分类案例优化

- 卷积层 API

- `CIFAR10` 数据集加载
